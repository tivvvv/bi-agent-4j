BI 智能体数据库表结构说明⽂档

1. ⽂档⽬的
本⽂件⽤于描述企业 BI 智能体系统使⽤的数据库表结构, 包括事实表, 维度表及其字段说明, 供后续 RAG 系统及 AI SQL Agent 使⽤.

2. 维度表结构

2.1 商品维度 (dim_product)
字段名 | 类型 | 主键 | 描述
product_id | bigint | YES | 商品唯⼀ ID
product_name | varchar(255) | | 商品名称
category_id | bigint | | 商品分类 ID
category_name | varchar(255) | | 商品分类名称
brand | varchar(255) | | 品牌
cost_price | decimal(10, 2) | | 成本价
retail_price | decimal(10, 2) | | 建议零售价

2.2 ⻔店维度 (dim_store)
字段名 | 类型 | 主键 | 描述
store_id | bigint | YES | ⻔店唯⼀ ID
store_name | varchar(255) | | ⻔店名称
province | varchar(100) | | 所在省份
city | varchar(100) | | 所在城市
address | varchar(255) | | ⻔店详细地址
open_date | date | | 开业⽇期

2.3 时间维度 (dim_date)
字段名 | 类型 | 主键 | 描述
date_id | int | YES | ⽇期 (yyyyMMdd)
date_value | date | | 实际⽇期
year | int | | 年份
quarter | int | | 季度 (1-4)
month | int | | ⽉份 (1-12)
day | int | | ⽇ (1-31)
week_num | int | | 周数
weekday | int | | 星期 (1-7)

2.4 顾客维度 (dim_customer)
字段名 | 类型 | 主键 | 描述
customer_id | bigint | YES | 顾客ID
customer_name | varchar(255) | | 顾客姓名
gender | varchar(10) | | 性别
age | int | | 年龄
city | varchar(100) | | 所在城市
province | varchar(100) | | 所在省份
week_num | int | | 周数
weekday | int | | 星期(1-7)

3. 事实表结构
3.1 销售事实表 (fact_sales)
字段名 | 类型 | 主键 | 描述
sales_id | bigint | YES | 销售记录唯⼀ ID
date_id | int | FK → dim_date(date_id) | 销售⽇期
customer_id | bigint | FK → dim_customer(customer_id) | 顾客ID
store_id | bigint | FK → dim_store(store_id) | 销售⻔店
product_id | bigint | FK → dim_product(product_id) | 销售商品
quantity | int | | 销售数量
sales_amount | decimal(10, 2) | | 销售⾦额(实收)
original_amount | decimal(10, 2) | | 应收⾦额(未折扣)
discount_amount | decimal(10, 2) | | 折扣⾦额

3.2 库存事实表 (fact_inventory)
字段名 | 类型 | 主键 | 描述
inventory_id | bigint | YES | 库存记录 ID
date_id | int | FK → dim_date(date_id) | ⽇期
store_id | bigint | FK → dim_store(store_id) | ⻔店
product_id | bigint | FK → dim_product(product_id) | 商品
stock_qty | int | | 库存数量
stock_value | decimal(10, 2) | | 库存⾦额
last_in_time | datetime | | 最近⼀次⼊库时间
last_out_time | datetime | | 最近⼀次出库时间

4. 表关系说明 (ER)
- fact_sales 关联 dim_product, dim_store, dim_date, dim_customer
- fact_inventory 关联 dim_product, dim_store, dim_date

5. ⽤途说明
这些表⽤于构建 BI 场景, 如: 
- 按⻔店, 商品, ⽇期进⾏销售分析
- GMV 统计
- 库存周转率分析
- 区域销售对⽐
- 热销商品分析
